# network-traffic-analyzer

# Overview
Network Traffic Analysis Tool written in Python for capturing live network packets. The project enables one to learn about the movement of data in the network and behavior of various protocols in packet form.
The project demonstrates techniques of SOCs that analyze network packets.

# Objectives
Capture live network packets
Identify network protocols (TCP, DNS, HTTP, etc.)
Understand real-time network communication
Build foundational SOC analysis skills

# Tools & Technologies
Python
Wireshark
pyshark (Python wrapper for Wireshark)

# How It Works
The tool connects to a network interface
Captures live packets
Extracts protocol information
Displays real-time traffic analysis in terminal

# How to Run
pip install pyshark
python traffic_analyzer.py
